# vue-casl-demo

This demo uses CASL to set permissions and enables corresponding features based on the current user.

## Project Setup
```sh
npm install
```

### Compile and Hot-Reload for Development
```sh
npm run dev
```

### Compile and Minify for Production
```sh
npm run build
```

### Use Plugin
```sh
@casl/ability
@casl/vue
vue-router
vue3
```
